# FormatSpec

Output format specification for structured data.

- JSON output: valid JSON, UTF-8 encoded
- YAML output: valid YAML 1.2, indentation 2 spaces
- Table output: pipe-delimited, header row required
- Timestamp format: ISO 8601 (YYYY-MM-DDTHH:MM:SSZ)
- Numeric precision: 2 decimal places for percentages
