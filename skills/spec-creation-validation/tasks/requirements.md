# Task: requirements

## Purpose

Extract and document requirements from the problem statement for the spec.

## Entry Criteria

- `research_artifact_path` is provided
- Problem statement is available

## Procedure

- [ ] 1. Read research artifact from `research_artifact_path`
- [ ] 2. Extract requirements from problem statement
- [ ] 3. Document requirements in spec format
- [ ] 4. Write requirements artifact to `./tmp/{issue-N}/artifacts/requirements.yaml`

## Exit Criteria

- Requirements artifact written with extracted requirements
- Artifact path returned

## Result Contract

| Field | Value |
|-------|-------|
| status | DONE | BLOCKED |
| finding_summary | "Extracted N requirements" |
| artifact_path | `./tmp/{issue-N}/artifacts/requirements.yaml` |
