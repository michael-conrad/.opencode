# Contract Index

Each contract template maps to a consuming pipeline step. Templates are named `{step}-{direction}-template.yaml`.

| Template | Direction | Consuming Step | Purpose |
|----------|-----------|----------------|---------|
| `research-input-template.yaml` | Input | research | Evidence artifacts input schema |
| `research-output-template.yaml` | Output | research | Evidence artifacts output schema |
| `readiness-input-template.yaml` | Input | readiness | Pipeline-readiness input schema |
| `readiness-output-template.yaml` | Output | readiness | Pipeline-readiness output schema |
| `structure-input-template.yaml` | Input | structure | Phase structure input schema |
| `structure-output-template.yaml` | Output | structure | Phase structure output schema |
| `solve-input-template.yaml` | Input | solve | Z3 solve input schema |
| `solve-output-template.yaml` | Output | solve | Z3 solve output schema |
| `write-input-template.yaml` | Input | write | Plan write input schema |
| `write-output-template.yaml` | Output | write | Plan write output schema |
| `revisit-input-template.yaml` | Input | revisit | Verification revisit input schema |
| `revisit-output-template.yaml` | Output | revisit | Verification revisit output schema |
| `validate-input-template.yaml` | Input | validate | Plan validation input schema |
| `validate-output-template.yaml` | Output | validate | Plan validation output schema |
| `audit-fidelity-input-template.yaml` | Input | audit-fidelity | Fidelity audit input schema |
| `audit-fidelity-output-template.yaml` | Output | audit-fidelity | Fidelity audit output schema |
| `audit-concern-input-template.yaml` | Input | audit-concern | Concern audit input schema |
| `audit-concern-output-template.yaml` | Output | audit-concern | Concern audit output schema |
| `completion-input-template.yaml` | Input | completion | Completion input schema |

| `create-input-template.yaml` | Input | create | Create pipeline input schema |
| `create-output-template.yaml` | Output | create | Create pipeline output schema |
