DESCRIPTION: Markdown tools dispatcher. Usage: ./.opencode/tools/md <action> [args...]
Actions: read, write, add, delete, new
Example: ./.opencode/tools/md read <file.md> [--section <header>]

Available actions:
  read                 — Read a section from a markdown file by header, or list all headers.
  write                — Replace the body of a section in a markdown file, preserving the header line.
  add                  — Insert a new section (header + body) into a markdown file.
  delete               — Delete a section (header line + body) from a markdown file by header.
  new                  — Create a new markdown file with an optional H1 title. Fails if file already exists.
